Methodology

Paired-end FASTQ files (R1 + R2) were processed through FastP for adapter trimming, quality filtering, and paired-end merging. Merged reads were aligned to the reference peptide library using Diamond BLASTX, yielding per-read amino acid and position assignments. Read counts were aggregated per amino acid per position per sample.

Two complementary analyses are performed: (1) Within-condition — an intercept-only binomial GLM (fit0) is fit at each position–AA pair; the log-odds coefficient reflects how frequently that amino acid is observed, shown in the sequence logo. (2) Pairwise — a full likelihood-ratio test (LRT) compares a condition-coded model (fit1) against the intercept-only model (fit0). The slope coefficient gives the log-odds ratio between conditions; positive = enriched in the test condition. All p-values are Benjamini–Hochberg corrected.
