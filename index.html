Capabilities
Everything in one pipeline

From raw sequencing files to statistically validated enrichment results and publication-ready figures — without leaving the browser.

Quality Control Pipeline

Automated adapter trimming, quality filtering, and paired-end merging via FastP. Per-sample QC metrics — pass rate, Q20/Q30, GC content — with pass/fail status.

FastP BBMerge Q30 tracking
Statistical Enrichment Modelling

Logistic regression with likelihood-ratio tests (LRT) identifies enriched amino acids at each position. Benjamini–Hochberg FDR correction controls the false discovery rate across all position–AA pairs.

Binomial GLM LRT BH FDR Log-odds
Interactive Sequence Logos

D3-rendered sequence logos with chemistry-based colouring (ggseqlogo scheme). Within-condition frequency logos and bidirectional pairwise logos — exportable as SVG for manuscripts.

D3.js SVG export Bidirectional Chemistry colours

Under the hood
Built for reproducibility & rigour
Django + REST API backend

All pipeline jobs managed through a RESTful API. Jobs are queued, tracked by UUID, and results serialised to a JSON field for fast retrieval.

R-backed statistical model

Enrichment analysis runs in R via rpy2 — binomial GLM with LRT at each position–AA pair, BH-corrected p-values, and log-odds coefficients for logo heights.

Vanilla JS SPA frontend

No framework overhead — a pure ES6 SPA with step-locked workflow, real-time job polling, and dynamic parameter panels driven by the pipeline registry.

D3-powered visualisations

Sequence logos use D3 v7 with per-letter SVG text scaling, chemistry-based colouring (ggseqlogo scheme), interactive filtering, and SVG export.
